# نظام توليد الأنسجة والمسارات الميكانيكية لآلات الـ CNC
## (CNC Texture Synthesis & G-Code Generator)

### 📝 وصف المشروع
مشروع تخرج يهدف إلى تحويل صور الزخارف الصغيرة إلى نماذج نسيجية واسعة النطاق (Texture Synthesis) وتوليد تعليمات برمجية (G-Code) للتحكم في آلات الحفر الرقمي. يعتمد النظام على معالجة الصور اليدوية لضمان دقة خريطة العمق (Depth Map).

### 🚀 المميزات الأساسية
- **معالجة مسبقة مخصصة:** تحويل RGB إلى Gray، تسوية Histogram، وعتبة Otsu يدوياً.
- **تحسين الأداء:** استخدام مكتبة `Numba` لتسريع العمليات الحسابية لتقترب من سرعة لغة C.
- **ذكاء النسيج:** خوارزمية SSD لمطابقة الرقع (Patches) وتوليد زخارف غير متناهية.
- **محرك التصنيع:** تحويل البيانات الرقمية إلى مسارات حفر فيزيائية دقيقة.

### 📁 هيكلية المجلدات
- `/core`: المحرك البرمجي لمعالجة الصور والنسيج.
- `/engine`: المسؤول عن توليد الـ G-Code.
- `/data`: مستودع الصور المدخلة والمخرجات.

### 🛠️ طريقة التثبيت والتشغيل

قم بتثبيت المكتبات المطلوبة:

Bash
pip install -r requirements.txt
لتشغيل النظام (المرحلة الحالية):

Bash
python main.py
